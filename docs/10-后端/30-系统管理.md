# 系统管理

> 核心系统管理功能：部门、区域、字典、菜单、系统配置、操作日志。模块路径：`spectra-core/system`。

## 实体总览

| Entity | 表说明 |
|---|---|
| `Department` | 组织架构（树形） |
| `Region` | 行政区划（树形） |
| `Menu` | 系统菜单（树形） |
| `DictGroup` | 字典分组 |
| `DictItem` | 字典项 |
| `Configured` | 配置表 |
| `SysConfig` | 系统参数 |
| `OperationLog` | 操作日志 |

## Department（部门）

树形结构，支持多级组织架构。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID | 主键 |
| `name` | String | 部门名称 |
| `parentId` | UUID | 父部门 ID |
| `sort` | Integer | 排序 |
| `leader` | String | 负责人 |
| `phone` | String | 联系电话 |
| `status` | Integer | 状态 |

## Region（区域）

行政区划数据，省/市/区县三级树形结构。

## Menu（菜单）

系统菜单树，支持多级嵌套。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID | 主键 |
| `name` | String | 菜单名称 |
| `parentId` | UUID | 父菜单 ID |
| `type` | Integer | 类型（目录/菜单/按钮） |
| `path` | String | 路由路径 |
| `component` | String | 前端组件路径 |
| `icon` | String | 图标 |
| `sort` | Integer | 排序 |
| `permission` | String | 权限标识 |
| `visible` | Boolean | 是否可见 |
| `status` | Integer | 状态 |

## DictGroup / DictItem（数据字典）

两级结构：DictGroup 是字典分组（如"性别"），DictItem 是字典项（如"男"/"女"）。

## Configured / SysConfig（系统配置）

- `Configured` — 配置表，支持 Key-Value 对
- `SysConfig` — 系统参数，支持按模块分类

## OperationLog（操作日志）

记录用户操作行为，用于审计追溯。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID | 主键 |
| `userId` | UUID | 操作用户 |
| `module` | String | 操作模块 |
| `action` | String | 操作类型 |
| `target` | String | 操作对象 |
| `ip` | String | 操作 IP |
| `userAgent` | String | 客户端信息 |
| `requestParams` | Text | 请求参数 |
| `responseResult` | Text | 响应结果 |
| `duration` | Long | 执行耗时(ms) |

## Controller

| Controller | 说明 |
|---|---|
| `DepartmentController` | 部门 CRUD + 树形查询 |
| `RegionController` | 区域查询 |
| `MenuController` | 菜单 CRUD + 树形查询 |
| `DictController` | 字典管理 |
| `ConfiguredController` | 配置表管理 |
| `ServiceMonitorController` | 服务监控（OSHI） |

## Service

| Service | 说明 |
|---|---|
| `DepartmentServiceImpl` | 部门管理 |
| `RegionServiceImpl` | 区域管理 |
| `MenuServiceImpl` | 菜单管理 |
| `DictServiceImpl` / `DictGroupServiceImpl` / `DictItemServiceImpl` | 字典管理 |
| `ConfiguredServiceImpl` | 配置表管理 |
| `SysConfigServiceImpl` | 系统参数管理 |
| `OperationLogServiceImpl` | 操作日志 |
| `ServiceMonitorServiceImpl` | 服务监控（基于 OSHI 6.9.1） |
