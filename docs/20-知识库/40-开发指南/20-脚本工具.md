# 脚本工具

> spectra-admin/document/scripts/ 中的实用脚本。

## 脚本清单

### toml2env.ps1

> 将 `.mise.local.toml` 的 `[env]` 段转换为 `.env` 文件。

```powershell
# 用法（默认路径）
.\toml2env.ps1

# 指定 TOML 文件
.\toml2env.ps1 -tomlFile "path/to/.mise.local.toml"
```

功能：
- 只处理 `[env]` 段
- 自动为包含空格或特殊字符的值加双引号
- 空字符串写成 `KEY=""`
- UTF-8 输出
- 生成完成后自动打开目录

### SSL 脚本

#### generate-cert.ps1

> 生成 localhost HTTPS 证书（依赖已安装的 CA）。

```powershell
# 不需要管理员权限
.\ssl\generate-cert.ps1
```

#### install-ca.ps1

> 安装 CA 证书到系统信任库。

```powershell
# 需要管理员权限
.\ssl\install-ca.ps1
```

#### uninstall-ca.ps1

> 从系统信任库卸载 CA 证书。

```powershell
# 需要管理员权限
.\ssl\uninstall-ca.ps1
```

## 使用场景

| 场景 | 使用脚本 |
|---|---|
| 首次配置本地环境 | `toml2env.ps1` |
| 本地 HTTPS 开发 | `ssl/install-ca.ps1` → `ssl/generate-cert.ps1` |
| 清理本地证书 | `ssl/uninstall-ca.ps1` |
