# 架构概览

> Spectra 系统精简架构，供 AI 快速建立项目认知。

## 系统架构

```
spectra-ui (Web)  ──┐
                    ├──→ spectra-admin (API :4004) ──→ PostgreSQL + Redis
spectra-app (移动端) ┘
```

## 技术栈

| 层级 | 技术 | 版本 |
|---|---|---|
| 后端框架 | Spring Boot | 4.1.0 |
| JDK | Java (Temurin) | 25 |
| 构建 | Maven (wrapper) | 3.9.12 |
| ORM | MyBatis-Plus | 3.5.15 |
| 数据库 | PostgreSQL | 18 |
| 缓存 | Redis + JetCache | 2.7.8 |
| AI | LangChain4j | 1.16.3 |
| 工作流 | Flowable | — |
| 前端框架 | Vue 3 + Vite | 8 (ui) / 5 (app) |
| UI 库 | Element Plus (ui) / TDesign (app) | — |
| 包管理 | pnpm | 11.0.9 |
| 运行时管理 | mise | — |

## 端口

| 服务 | 端口 |
|---|---|
| spectra-admin (API) | 4004 |
| spectra-ui (Web) | 5173 |
| PostgreSQL | 5432 |
| Redis | 6379 |

## 目录结构

```
spectra/
├── spectra-admin/          ← 后端 API 服务
│   ├── spectra-config/     ← 统一配置文件
│   ├── spectra-common/     ← 公共模块（DTO/Entity基类/工具类）
│   ├── spectra-framework/  ← 框架层（MVC/JSON/缓存/Redis/MyBatis）
│   ├── spectra-modules/    ← 业务模块
│   │   ├── spectra-core/   ← 用户/权限/系统管理
│   │   ├── spectra-oa/     ← OA 办公
│   │   ├── spectra-upload/ ← 文件上传
│   │   ├── spectra-workflow/ ← 工作流
│   │   └── spectra-ai/     ← AI 集成
│   ├── spectra-starter/    ← 自动配置 Starter
│   └── spectra-launch/     ← 启动入口
├── spectra-ui/             ← Web 管理后台
├── spectra-app/            ← 移动端/H5/微信小程序
└── logicflow-flowable/     ← BPMN 流程建模插件
```
