# 配置清单

> spectra-admin 环境变量与配置项完整列表。

## 数据库

| 环境变量 | 说明 | 示例 |
|---|---|---|
| `DB_URL` | 数据库连接 URL | `jdbc:postgresql://127.0.0.1:5432/devops00_spectra_db` |
| `DB_USERNAME` | 数据库用户名 | `postgres` |
| `DB_PASSWORD` | 数据库密码 | `xxx` |

## Redis

| 环境变量 | 说明 | 示例 |
|---|---|---|
| `REDIS_HOST` | Redis 地址 | `127.0.0.1` |
| `REDIS_PORT` | Redis 端口 | `6379` |
| `REDIS_DB` | Redis 数据库 | `0` |
| `REDIS_PASSWORD` | Redis 密码 | `xxx` |

## 服务器

| 环境变量 | 说明 | 示例 |
|---|---|---|
| `SERVER_PORT` | 服务端口 | `4004` |
| `DEFAULT_PASSWORD` | 新增/重置用户默认密码 | `admin123` |

## SSL

| 环境变量 | 说明 | 示例 |
|---|---|---|
| `SSL_PASSWORD` | SSL 密码 | `xxx` |
| `SSL_TYPE` | SSL 类型 | `PKCS12` |
| `SSL_ALIAS` | SSL 别名 | `tomcat` |

## S3 存储

| 环境变量 | 说明 | 示例 |
|---|---|---|
| `S3_ENDPOINT` | S3 端点 | `https://xxx` |
| `S3_ACCESS_KEY` | S3 访问密钥 | `xxx` |
| `S3_SECRET_KEY` | S3 秘密密钥 | `xxx` |
| `S3_BUCKET` | S3 存储桶 | `xxx` |
| `S3_REGION` | S3 区域 | `xxx` |

## 接口加解密

| 环境变量 | 说明 | 默认值 |
|---|---|---|
| `SPECTRA_SYSTEM_SM_ENABLED` | 是否启用接口加解密 | `false` |
| `SPECTRA_SYSTEM_SM_PUBLIC_KEY` | RSA 公钥 (Base64) | — |
| `SPECTRA_SYSTEM_SM_PRIVATE_KEY` | RSA 私钥 (Base64) | — |

## 前端配置

### spectra-ui

| 环境变量 | 说明 | 默认值 |
|---|---|---|
| `VITE_API_URL` | 后端 API 基础地址 | `https://127.0.0.1:4004/` |
| `VITE_WEB_TITLE` | 网站标题 | `光谱平台` |
