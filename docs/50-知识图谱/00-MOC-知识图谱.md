# 知识图谱导航

> 面向 AI Agent 的结构化速查数据，用于快速建立项目上下文。

## 速查入口

| 文档 | 用途 | 何时使用 |
|---|---|---|
| [[01-架构概览]] | 系统架构、技术栈、端口 | 新会话初始化 |
| [[02-模块清单]] | 所有模块职责与依赖 | 理解模块关系 |
| [[03-实体字典]] | 25 个 Entity 字段速查 | 修改数据库/实体 |

## 使用方式

AI Agent 在新会话开始时：
1. 读取 [[01-架构概览]] 建立全局认知
2. 根据任务读取对应速查表
3. 使用 CodeGraph 索引查询具体代码实现

## 与知识库的区别

| 维度 | 知识图谱 | 知识库 |
|---|---|---|
| 受众 | AI Agent | 人类开发者 |
| 格式 | 表格化、扁平化 | 叙述式、分层 |
| 深度 | 速查概览 | 详细实现 |
| 更新频率 | 代码变更时同步 | 功能完成时更新 |
