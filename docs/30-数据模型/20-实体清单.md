# 实体清单

> spectra-admin 全部实体类完整字典。共 25 个 Entity（不含 BaseEntity）。

## 公共基类

### BaseEntity

> 所有实体的公共父类。
> 路径：`spectra-common/src/main/java/com/devops00/spectra/common/base/BaseEntity.java`

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID (String) | 主键，UUID v7 |
| `createdBy` | String | 创建人 ID |
| `createdAt` | LocalDateTime | 创建时间 |
| `updatedBy` | String | 更新人 ID |
| `updatedAt` | LocalDateTime | 更新时间 |
| `deleted` | LocalDateTime | 软删除时间（NULL=未删除） |
| `version` | Integer | 乐观锁版本号 |

## OA 模块

### Asset / Attendance / Calendar / Contact / Contract / Document / Notice / Report

> 路径：`oa/{module}/javabean/entity/`

各 OA 子模块的实体类，均继承 BaseEntity。

### Meeting / MeetingParticipant / MeetingRecord

> 路径：`oa/meeting/javabean/entity/`

- **Meeting** — 会议基本信息
- **MeetingParticipant** — 参会人员关联
- **MeetingRecord** — 会议纪要
