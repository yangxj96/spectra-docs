## UUID v7 兼容

PostgreSQL 18 原生支持 `uuidv7()` 函数（注意：无下划线，非 `uuid_v7()`）。

若精简版镜像未自带扩展，可内建兼容函数：

```sql
CREATE FUNCTION domain_core.uuid_v7() RETURNS uuid
   LANGUAGE sql
   AS $$
SELECT
upper(
(
lpad(to_hex((extract(epoch from clock_timestamp()) * 1000)::bigint), 12, '0') ||
lpad(to_hex((random() * 2^12)::int), 3, '0') ||
'7' ||
lpad(to_hex((random() * 2^62)::bigint), 16, '0')
)::text
)::uuid;
$$;
```

> 如果 `uuid_v7()` 调用失败，检查实际函数名是否为 `uuidv7()`。

## pg_dump 导出/导入

### 导出指定 schema 的结构和数据

```bash
pg_dump -h localhost -U postgres -d spectra_db \
  -n domain_core \
  --section=pre-data --section=data \
  --no-owner --no-privileges \
  > domain_core.sql
```

### 压缩导出

```bash
# gzip 方式
pg_dump -h localhost -U postgres -d devops00_spectra_db \
  -n spectra_core --section=pre-data --section=data \
  --no-owner --no-privileges \
  | gzip > core.sql.gz

# 还原
gunzip -c core.sql.gz | psql -h localhost -U postgres -d devops00_spectra_db
```

### 自定义格式导出（推荐）

```bash
# 导出
pg_dump -h localhost -U postgres -d devops00_spectra_db \
  -n spectra_core -Fc -Z 9 --no-owner --no-privileges \
  -f core.dump

# 还原
pg_restore -h localhost -U postgres -d devops00_spectra_db core.dump

# Docker 容器直接导入
docker exec -i spectra-postgres pg_restore \
  -U postgres -d devops00_spectra_db < /data/docker/spectra/core.dump
```

## 数据库初始化

### 执行顺序

```bash
# 1. 创建核心表结构
psql -h localhost -U postgres -d spectra_db -f 20-知识库/30-数据模型/30-核心建表SQL.sql

# 2. 创建 OA 表结构
psql -h localhost -U postgres -d spectra_db -f 20-知识库/30-数据模型/31-OA建表SQL.sql

# 3. 创建 RAG 表结构（需要先启用 pgvector 扩展）
psql -h localhost -U postgres -d spectra_db -f 20-知识库/30-数据模型/32-RAG建表SQL.sql

# 4. 导入初始化数据（角色/权限/菜单/部门/字典/用户）
psql -h localhost -U postgres -d spectra_db -f 20-知识库/30-数据模型/33-初始化数据.sql

# 5. 恢复行政区划数据（sys_region 表，665k 行）
pg_restore -h localhost -U postgres -d spectra_db --data-only --table=spectra_core.sys_region 20-知识库/30-数据模型/34-行政区划数据.dump
```

### Docker 容器导入

```bash
# 复制文件到容器
docker cp 20-知识库/30-数据模型/ spectra-postgres:/tmp/

# 在容器内执行
docker exec -i spectra-postgres psql -U postgres -d spectra_db -f /tmp/30-数据模型/30-核心建表SQL.sql
docker exec -i spectra-postgres psql -U postgres -d spectra_db -f /tmp/30-数据模型/31-OA建表SQL.sql
docker exec -i spectra-postgres psql -U postgres -d spectra_db -f /tmp/30-数据模型/32-RAG建表SQL.sql
docker exec -i spectra-postgres psql -U postgres -d spectra_db -f /tmp/30-数据模型/33-初始化数据.sql
docker exec -i spectra-postgres pg_restore -U postgres -d spectra_db --data-only --table=spectra_core.sys_region /tmp/30-数据模型/34-行政区划数据.dump
```
