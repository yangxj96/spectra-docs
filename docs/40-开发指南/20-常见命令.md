# 常见命令

> 日常开发中常用的命令速查。

## spectra-admin（后端）

```bash
# 从 spectra-admin/ 目录执行

# 编译打包（跳过测试）
./mvnw clean package -DskipTests

# 只编译，不打包
./mvnw clean compile

# 运行测试
./mvnw test

# 启动 API 服务器
./mvnw spring-boot:run -pl spectra-launch

# 只编译单个模块
./mvnw clean compile -pl spectra-core -am
# -pl: 指定模块
# -am: also-make，同时编译依赖的模块
```

## spectra-ui（Web 管理后台）

```bash
# 从 spectra-ui/ 目录执行

pnpm install         # 安装依赖
pnpm start           # 启动开发服务器（:5173），自动 format+lint+typecheck
pnpm build           # 生产构建
pnpm preview         # 预览生产构建
pnpm lint            # ESLint 检查
pnpm typecheck       # TypeScript 类型检查
pnpm format          # Prettier 格式化
```

## spectra-app（移动端）

```bash
# 从 spectra-app/ 目录执行

pnpm install         # 安装依赖
pnpm start           # H5 开发，自动 typecheck+lint+format
pnpm dev:mp-weixin   # 微信小程序开发
pnpm build           # H5 构建
pnpm build:mp-weixin # 微信小程序构建
```

## Git 约定

```bash
# 提交格式（Conventional Commits）
# type(scope): 中文描述

# 示例
git commit -m "feat(core): 新增用户批量导入功能"
git commit -m "fix(security): 修复 Token 刷新异常"
git commit -m "refactor(upload): 重构分片上传逻辑"

# 常见 type
# feat     — 新功能
# fix      — 修复 Bug
# refactor — 重构
# docs     — 文档更新
# style    — 代码格式
# test     — 测试
# chore    — 构建/工具变更
```

## 常用 scope

| scope | 说明 |
|---|---|
| `core` | 核心模块 |
| `oa` | OA 模块 |
| `upload` | 上传模块 |
| `workflow` | 工作流模块 |
| `ai` | AI 模块 |
| `security` | 安全认证 |
| `framework` | 框架层 |
| `ui` | Web 管理后台 |
| `app` | 移动端 |
