# 环境搭建

> Spectra 项目完整开发环境搭建指南。

## 前置依赖

| 依赖 | 版本 | 安装方式 |
|---|---|---|
| JDK | 25 (Temurin) | 通过 mise 或手动安装 |
| Node.js | 24.14.0 | 通过 mise 管理 |
| pnpm | 11.0.9 | 通过 mise 管理 |
| Maven | 3.9.12 | 项目自带 `mvnw` wrapper |
| PostgreSQL | 18 | Docker 或本地安装 |
| Redis | — | Docker 或本地安装 |

## mise 配置

项目使用 [mise](https://mise.jdx.dev/) 管理运行时版本。

根目录 `mise.toml` 定义全局工具版本。`spectra-admin/.mise.local.toml` 定义后端专用配置。

完整环境变量清单参见 `spectra-admin/README.md` 和 `.mise.local.toml.example`：

| 分组 | 变量 | 说明 |
|---|---|---|
| 数据库 | `DB_URL`、`DB_USERNAME`、`DB_PASSWORD` | PostgreSQL 连接 |
| 服务 | `SERVER_PORT`、`DEFAULT_PASSWORD` | 端口与默认密码 |
| Redis | `REDIS_HOST`、`REDIS_PORT`、`REDIS_DB`、`REDIS_PASSWORD` | 缓存连接 |
| SSL | `SSL_PASSWORD`、`SSL_TYPE`、`SSL_ALIAS` | 本地 HTTPS |
| S3 | `S3_ENDPOINT`、`S3_ACCESS_KEY`、`S3_SECRET_KEY`、`S3_BUCKET`、`S3_REGION` | 对象存储 |
| AI | `AI_KEY`、`AI_BASE_URL`、`AI_MODEL` | AI 服务 |
| RAG | `RAG_KEY`、`RAG_BASE_URL`、`RAG_MODEL` | 检索增强生成 |

```toml
# spectra-admin/.mise.local.toml （从 .example 复制并修改）
[tools]
java = "temurin-25.0.2+10.0.LTS"

[env]
# 数据库
DB_URL = "jdbc:postgresql://localhost:5432/devops00_spectra_db"
DB_USERNAME = "postgres"
DB_PASSWORD = "xxx"
# 服务
SERVER_PORT = "8080"
DEFAULT_PASSWORD = "admin123"
# Redis
REDIS_HOST = "127.0.0.1"
REDIS_PORT = "6379"
REDIS_DB = "0"
REDIS_PASSWORD = "XXX"
# SSL（可选）
SSL_PASSWORD = "xxx"
SSL_TYPE = "PKCS12"
SSL_ALIAS = "tomcat"
# S3（可选）
S3_ENDPOINT = ""
S3_ACCESS_KEY = ""
S3_SECRET_KEY = ""
S3_BUCKET = ""
S3_REGION = ""
# AI（可选）
AI_KEY = ""
AI_BASE_URL = ""
AI_MODEL = ""
# RAG（可选）
RAG_KEY = ""
RAG_BASE_URL = ""
RAG_MODEL = ""
```

## 初始化步骤

### 1. 配置后端

```bash
# 进入 spectra-admin 目录
cd spectra-admin

# 复制环境配置模板
copy .mise.local.toml.example .mise.local.toml

# 编辑 .mise.local.toml，填入实际的数据库/Redis/S3/AI 凭据
```

### 2. 配置前端

```bash
# spectra-ui（默认已配置，指向 localhost:4004）
cd spectra-ui
# .env 已存在，如需修改：
# VITE_API_URL=https://127.0.0.1:4004/

# spectra-app（默认已配置）
cd spectra-app
# .env.development 已存在：
# VITE_API_BASE_URL=https://127.0.0.1:4004
```

### 3. 启动数据库和 Redis

确保 PostgreSQL 和 Redis 服务已启动。

### 4. 启动后端 API

```bash
cd spectra-admin
./mvnw spring-boot:run -pl spectra-launch
# API 运行在 https://127.0.0.1:4004
```

### 5. 启动前端

```bash
# Web 管理后台
cd spectra-ui
pnpm install
pnpm start           # → http://localhost:5173

# 移动端 H5
cd spectra-app
pnpm install
pnpm start           # → http://localhost:xxxx
```
