# OA 模块

> 办公自动化：资产/考勤/日历/通讯录/合同/文档/会议/公告/报表。

## 模块简介

`spectra-oa` 是 Spectra 平台中的办公自动化（OA）模块，用于提供企业内部日常办公相关的基础能力。

OA 模块本身不负责复杂审批流程，涉及审批的业务通过 **workflow 工作流模块** 处理，实现流程与业务解耦。

## 目录结构

```
spectra-oa/
├── notice      通知公告
├── calendar    日历管理
├── meeting     会议管理
├── document    文档中心
├── attendance  考勤管理
├── asset       资产管理
├── contact     通讯录
├── report      统计报表
└── contract    合同管理
```

## 子模块功能

### notice（通知公告）

- 发布公告、通知分类管理
- 通知阅读记录、置顶
- 定时发布、附件支持
- 场景：公司公告、部门通知、系统通知

### calendar（日历管理）

- 个人/团队日程管理
- 日程提醒、周/月视图
- 场景：工作计划、项目节点提醒、会议安排

### meeting（会议管理）

- 会议创建、会议室预约
- 参会人员管理、会议通知
- 会议记录、会议附件
- 场景：项目会议、周会、技术评审

### document（文档中心）

- 文档上传、分类、权限控制
- 文档版本管理、检索、预览
- 场景：公司制度、技术文档、项目资料

### attendance（考勤管理）

- 打卡记录、请假/出差/加班申请
- 考勤统计、考勤报表
- 审批操作通过 workflow 模块处理

### asset（资产管理）

- 资产登记、分类、领用/归还
- 资产维修、报废
- 场景：办公设备、IT设备、固定资产

### contact（通讯录）

- 部门结构、员工信息
- 联系方式、组织架构展示
- 快速检索

### report（统计报表）

- 考勤/资产/合同/工作流统计
- 自定义报表
- 支持按时间/部门/用户统计

### contract（合同管理）

- 合同登记、分类、附件
- 合同到期提醒、合同审批
- 审批流程通过 workflow 模块完成

## 模块依赖

```
spectra-oa → spectra-core（基础能力）
           → spectra-workflow（审批流程）
```

## 工作流集成

OA 中需要审批的业务通过 workflow 模块处理：

```
业务模块 → workflow → 审批 → 更新业务数据
```

涉及审批的业务：
- 请假申请、加班申请
- 资产申请、合同审批
